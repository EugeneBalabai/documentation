import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

To install Tolgee run:

<Tabs
    defaultValue="npm"
    values={[
        {label: 'npm', value: 'npm',},
        {label: 'yarn', value: 'yarn'},
    ]
    }>
<TabItem value="npm">

```shell
npm install @tolgee/core
```

</TabItem>
<TabItem value="yarn">

```shell
yarn add @tolgee/core
```

</TabItem>
</Tabs>

To install UI bundle for in-context translating you will also need to install package `@tolgee/ui`.

<Tabs
    defaultValue="npm"
    values={[
        {label: 'npm', value: 'npm'},
        {label: 'yarn', value: 'yarn'},
    ]
    }>
<TabItem value="npm">

```shell
npm install @tolgee/ui
```

</TabItem>
<TabItem value="yarn">

```shell
yarn add @tolgee/ui
```

</TabItem>
</Tabs>

After installing packages you can use it like this:

```js
import { Tolgee, DevTools } from '@tolgee/web';

const tg = Tolgee().use(DevTools()).init({
  apiKey: 'your_api_key',
  apiUrl: 'https://app.tolgee.io',
});

tg.run();
```

Then you can use Tolgee instance the same way as described in [translating imperatively](../get_started/translating_imperatively).
