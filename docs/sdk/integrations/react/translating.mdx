There are several ways how to translate a text using our React integration.

## The [`T`](./api#t-component) component

First way is the [`<T>`](./api#t-component) component. You can use it like this:

```jsx
import {T} from "@tolgee/react";
...
<T keyName="translation_key" />
```

To pass variable parameters just assign params property.

```jsx
<T keyName="text_with_params" params={{ param: 'value' }} />
```

> Read more in [Formatting](/docs/sdk/formatting) section

### Providing default value

When `keyName` property is provided, then children prop of [`T`](./api#t-component) component is considered as default value.

```jsx
<T keyName="translation_key">This is default value.</T>
```

The default value will be rendered when no translation is provided in current nor fallback language.

## `useTranslate` hook

Other option how to translate is using of [`useTranslate`](./api#hook-usetranslate) hook. You can use it this way:

```js
import {useTranslate} from "@tolgee/react";
...
const {t} = useTranslate();
...
t("key_to_translate")
```

Function [`t`](./api#function-t) also accepts parameters object as second parameter:

```js
t('key_to_translate', { param: 'value' });
```

For more complex codebase showing usage of Tolgee React integration check
[React example application](https://github.com/tolgee/react-sampleapp).

## Message format

By default Tolgee is without any message formatter. You can use [`FormatSimple`](/docs/sdk/api/core_package/format-simple) or [`FormatIcu`](/docs/icu-message-format)
